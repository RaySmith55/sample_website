Important HTML Tags



This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.


The tables below don't include all HTML tags, but they do list many of the most important ones.


Block Elements

Tag Name Purpose
h1–h6 Headings Include a heading (levels 1–6)
p Paragraph Include a paragraph of text
table Table Include a table
tr Table Row Include a row of data
th Table Header Make a table header
td Table Data Include a table data cell
header Header Define content as being in the page header
div Div Set page divisions apart
ol Ordered List Create a list of items, numbered
ul Unordered List Create a list of items, bullet pointed


Inline Elements

Tag Name Purpose Example Result
em Emphasized Make emphasized text <em>technical sophistication</em> technical sophistication
strong Strong Make strong text <strong>take control of your computer</strong> take control of your computer
a Anchor Anchor an element to a certain location <a href="http://google.com/" target="_blank">Hyperlink</a> Hyperlink
img Image Insert an image onto the page <img src="images/small_twitter_logo.png" alt="Test image"></img>
code Code Make text format in code style <code>Test code</code> Test code
span Span Declare text as being in a division, for styling <span>Call me Ishmael.</span> Call me Ishmael.
li List Declare text as being in a list item <li>Vengeful whale</li>
